# Project Documentation

## Project Goal

Develop a virtual training environment for autonomous drones using ROS2 Jazzy and Gazebo Harmonic.

The drone is repeatedly tested through simulation sessions involving:

- Waypoint navigation
- Obstacle avoidance
- Mapping
- Landing zone approach
